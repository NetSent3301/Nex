# Custom Agent Creator

## Description
Create a specialized AI agent for a specific task. Define its purpose, tools, and behavior.

## Usage
Use this skill when you need to create a focused agent for repeated tasks.

## Instructions
1. Define the agent's name and primary objective
2. Specify which tools the agent can use
3. Set the agent's behavior and tone
4. Define the expected output format
5. Register the agent for reuse
